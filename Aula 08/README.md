# Auditoria Técnica: Acervo Pica-Pau
Projeto de Aplicações para Internet - UNIPÊ 2026.

## 1. Definição do Problema
* **Contexto**: Plataforma de auditoria e catalogação de masters de dublagens clássicas.
* **Público-alvo**: Pesquisadores de mídia e entusiastas da preservação audiovisual.
* **Dor**: Dificuldade em validar a integridade técnica de lotes de dublagem (AIC, BKS, etc.) de forma centralizada.
* **Critério de Sucesso**: O usuário consegue validar um lote e conferir a mídia com acessibilidade em uma única interface.
